# Product Brief: tac'tile

**Date:** 2026-03-08
**Author:** rboniface
**Version:** 1.0
**Project Type:** Game
**Project Level:** 3 (Complex)

---

## Executive Summary

tac'tile is a roguelite tactical RPG that combines Terra Battle's signature combat system (grid-based movement, pincer attacks, chaining, and elemental mechanics) with run-based roguelite progression. Players send their units on runs to gather loot and currencies, then invest those resources to permanently improve their characters. The game targets gacha enthusiasts, tactical strategy fans, and theory crafters who enjoy optimizing builds and progression systems.

---

## Problem Statement

### The Problem

Terra Battle is dead, leaving its signature tactical combat system without an active home. Meanwhile, roguelites offer engaging run variety but lack the gacha-style character progression that creates long-term investment and attachment.

### Why Now?

AI-assisted development tools are making it feasible for indie developers to build complex tactical RPGs that previously required larger teams and budgets.

### Impact if Unsolved

Players continue replaying dead games with declining communities or settle for experiences that only partially satisfy their tactical and progression cravings.

---

## Target Audience

### Primary Users

Mobile gamers aged 25-40 who play gacha games (Arknights, Friends & Dragons, Fire Emblem) and roguelites (Slay the Spire). They're familiar with both systems and actively seek games that combine them.

### Secondary Users

None - focused exclusively on mobile-first tactical gaming audience.

### User Needs

1. Deep tactical combat that rewards mastery and positioning
2. Long-term character progression that creates investment and attachment
3. Replayability through run variety and build experimentation

---

## Solution Overview

### Proposed Solution

Grid-based tactical combat runs where players earn loot and currency to permanently upgrade their characters and summon new units through gacha.

### Key Features

**Combat System:**

- Pincer attacks

- Elemental system
- Character-specific skills

**Run Structure:**
- Procedurally generated maps
- Node-based path selection
- Boss encounters

**Progression:**
- Character stat growth

### Value Proposition

tac'tile is the only game that combines Terra Battle's deep tactical combat with roguelite run structure AND gacha-style character progression - three systems players currently have to choose between.

## Risks and Mitigation

- **Risk: Burnout** (Solo dev, aggressive timeline)
  - Likelihood: High
  - Mitigation: Set sustainable work hours, build buffer into timeline, celebrate milestones, know minimum viable launch criteria

- **Risk: Scope Creep** (Feature creep during development)
  - Likelihood: Medium
  - Mitigation: Keep "out of scope" list visible, use "backlog for later" document, any new feature = trade equal effort, regular scope reviews

- **Risk: Monetization Balance** (Gacha economy difficulty)
  - Likelihood: High
  - Mitigation: Study successful gacha benchmarks, start conservative, soft launch for data, build analytics from day 1, consider community/consultant feedback

- **Risk: Unit Balance** (Tactical combat requires tight balance)
  - Likelihood: High
  - Mitigation: Start with small roster (8-12 units), use spreadsheet/math models, automated combat testing, beta testers for balance feedback, expect post-launch rebalancing
